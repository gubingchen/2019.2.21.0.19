# 编码问题
- 为什么需要编码问题
    - 本质上计算机只能识别01代码
    - 如何用一长串01代码表示复杂的信息
- 编码简史
    - 二进制
        - bit： 一个0或1的二进制数字
        - byte：八个01代码，字节
    - 第一阶段：ASCII
    - 第二阶段：百花齐放，GB2312，GBK，BIG5，Latin1，JIS
        - Latin1：兼容欧洲大多数语言
        - 中国：GBxxxx
        - 韩国台湾：BIG5
        - 日本：JIS
        - ANSI-MBCS（Multi-bytes character set,多字节字符集）
    - 第三个阶段：Unicode（ISO）
